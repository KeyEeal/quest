# Audio Assets

This folder is intentionally empty by default.

The Phantom route is configured to look for:

```text
assets/audio/phantom-placeholder.mp3
```

Only add audio that you own, created yourself, or are legally allowed to use and distribute with this static site. Do not commit copyrighted soundtrack audio by default. If the configured audio file is missing, the site shows a graceful placeholder and uses the musical cipher fallback instead.
